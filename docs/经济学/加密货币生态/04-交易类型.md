# 交易类型

## 一、现货交易

同股市买卖一样。即买即得，买卖双方直接交换数字资产。

## 二、杠杆交易

杠杆交易允许交易者使用借来的资金进行交易，从而以更小的本金控制更大的交易量。交易者可以利用杠杆来放大收益，但同样也会放大损失。

仍旧是币币交易。
